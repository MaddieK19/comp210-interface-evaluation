Skyrim Evaluation

The screen based interface that will be evaluated is Skyrim on PC.

The video above shows many of Skyrim's menu screens being navigated on PC using a mouse.

Skyrim uses the same UI for PC and console and as an Xbox controller can be used on PC some parts may be relevant to the Xbox version of the game as well as the PC version. Many reviews for the PC version suggest that while this UI may be easy to use on console it can be unintuitive on PC. For example PC gamer describes Skyrim's UI on PC as "terrible" and say that mods makes the UI "more pleasant" and "useful".

On the modding website Nexus the two most endorsed mods for Skyrim are both UI mods. The first overhauls most of Skyrim's existing menu screens and adds in features such as character status and mod management menus. SkyUI describes itself as a "PC friendly UI mod for Skyrim" suggesting the creators do not think the UI is already "PC friendly". The second alters the map screens to show road locations making the game easier to navigate.
