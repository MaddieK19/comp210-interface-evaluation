Assignment One

The screen based interface that will be evaluated is Skyrim on PC. I've chosen this game because a common complaint about Skyrim on PC is that the UI isn't tailored to PC it's the same as the console version. This means the menus can be unintuitive to navigate. A further reason for evaluating Skyrim's interface is that currently the most popular Skyrim mod on nexus is SkyUI which is a mod that overhauls many of the menu screen in the existing interface.
